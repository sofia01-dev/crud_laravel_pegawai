# CRUD Pegawai Laravel

## 📌 Deskripsi
Aplikasi CRUD (Create, Read, Update, Delete) data pegawai menggunakan Laravel, PHP, dan MySQL.

## ⚙️ Teknologi
- PHP
- Laravel
- MySQL
- XAMPP

## 🚀 Cara Instalasi

1. Clone repository
git clone https://github.com/sofia01-dev/crud_laravel_pegawai.git

2. Masuk ke folder
cd crud_laravel_pegawai

3. Install dependency
composer install

4. Copy file .env
cp .env.example .env

5. Generate key
php artisan key:generate

6. Setting database di .env
DB_DATABASE=crud_pegawai
DB_USERNAME=root
DB_PASSWORD=

7. Jalankan migration
php artisan migrate

8. Jalankan server
php artisan serve

## 🌐 Akses
http://127.0.0.1:8000/pegawai
